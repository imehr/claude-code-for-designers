This release adds a full Dynamic Workflows section to Chapter 7 and republishes all public artifacts: HTML, paged HTML, PDF, EPUB, and cover image.

### Content Changes

**Dynamic Workflows for Designers.** Added a complete section explaining Claude Code Dynamic Workflows as a way to move multi-stage design work out of one chat and into an inspectable orchestration script. The update includes the bundled `/deep-research` workflow, `/workflows`, project and personal workflow locations, a subagents/skills/workflows comparison, a component-library audit workflow, reusable design-team workflow prompts, detailed design use cases, and cost/safety rules for model routing and permissions.
